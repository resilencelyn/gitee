F6 提供了[默认箭头和内置箭头](/zh/docs/manual/middle/elements/edges/arrow)。当这些箭头不满足条件时，可以[自定义箭头](/zh/docs/manual/middle/elements/edges/arrow#自定义箭头)。

## 何时使用

为内置边和自定义边配置起始端箭头和结束端箭头。
