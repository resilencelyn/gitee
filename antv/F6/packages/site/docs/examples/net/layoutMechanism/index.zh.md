F6 为用户提供了布局参数动态切换、布局方法动态切换、数据动态切换、子图布局、布局时机监听、自定义布局等布局相关等机制。

## 使用指南

- 代码演示 1 ：控制子图（红色节点）使用经典力导向算法进行子图布局。
- 代码演示 2 ：同时设置不同的子图布局。
- 代码演示 3 ：布局数据可以动态切换。
- 代码演示 4 ：F6 提供了自定义布局机制，可以随心所欲定制自己的布局。
- 代码演示 5 ：F6 开放了布局的监听时机 —— `beforelayout` 与 `afterlayout`。
