F6 自定义节点与边机制允许用户对树图视觉样式进行定制。

## 使用指南

通过结合自定义元素，可以实现较为复杂的树可视化。

- 代码演示 1： 自定义树图中的节点及边，支持增加和删除节点；
- 代码演示 2 ：为边增加文本标签。
- 代码演示 3 ：自定义带有收缩/扩展按钮的节点。
