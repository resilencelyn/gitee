通过自定义边，可以实现边的动画效果。

### 使用指南

下面代码演示了三种通过自定义边实现的边动画：

- 圆点在沿着线运动；
- 虚线运动的效果；
- 线从无到有的效果。

F6 中更多关于动画的内容，请参考[动画文档](/zh/docs/manual/middle/animation)。
