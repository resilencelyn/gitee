通过 `changeData` 方法不断地改变节点位置，以达到动画的效果。

### 使用指南

对于一些简单的动画场景，可以通过这种方式来实现。稍复杂的场景，不建议使用这种方式。

F6 中更多关于动画的内容，请参考文档 [动画](/zh/docs/manual/middle/animation)。
